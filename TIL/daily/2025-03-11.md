Git 저장소 초기화

```jsx
git init
```

← 현재 디렉토리를 Git 저장소로 초기화

```jsx
git add .      // 모든 변경 사항을 추가
```

```jsx
git commit -m "메세지"  // 변경 사항을 -m 옵션을 사용하여 커밋
```

```jsx
git push       // 저장소에 업로드
```

### **gitignore**

- 파일이나 폴더를 git에서 추적하지 않게 설정하는 파일.
- 환경 설정 파일이나 로그 파일 등을 무시할 때 사용.
