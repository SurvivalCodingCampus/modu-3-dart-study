## 객체를 도출하는 순서

- 객체가 가지고 있을만한 속성이나 동작을 생각 해 종류와 내용을 정의
- 역할과 책임을 구분

## Class

- 사용자 지정 유형
- 객체를 생성하기 위한 틀

## Instance

- 클래스를 기반으로 런타임에 실제로 메모리에 생성된 객체

## Optional Parameter

- 다트에서 옵셔널 파라미터는 대괄호로 감싸서 사용함

## Initialize List

- `:`은 초기화 리스트를 의미
- 생성자 바디가 실행되기 전에 필드를 초기화하는 코드 작성 가능
- 생성자 본문 전에 초기화 해야하는 경우 사용
- `final` 또는 `late final` 변수는 생성자 본문에서 초기화 불가능
