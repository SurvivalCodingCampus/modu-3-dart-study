# 과제 설명

## 캡슐화

- wand 클래스에서 setter로 다음과 같은 조건 추가
    - 지팡이의 이름은 3문자 이상이어야 함
    - 지팡이의 마력은 0.5이상 100.0 이하여야 함.

- wizard 클래스에서 setter로 다음과 같은 조건 추가
    - 마법사의 지팡이는 null일 수 없음.
    - 마법사의 이름은 3문자 이상이어야 함.
    - 마법사의 MP는 0 이상이어야함.
    - 마법사의 HP가 음수가 되면 대신 0으로 설정함.

- setter의 타당성 검사를 인스턴스 생성 시에도 적용하기 위해 생성자의 초기화 리스트에도 동일한 throw Exception 조건 추가
